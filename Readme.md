## Классификатор изображений диких животных для мониторинга

### Описание:

Этот проект представляет собой классификатор изображений, разработанный для автоматизации процесса категоризации фотографий диких животных, полученных с фотоловушек. Проект призван помочь исследователям в мониторинге популяций диких животных, облегчая рутинную работу по анализу огромного количества данных.

### Цель:

Цель проекта - создание модели машинного обучения, способной классифицировать изображения диких животных на заданные группы, организованные по принципу схожести видов.

### Реализация:

- Библиотека: torchvision (PyTorch)
- Набор данных: предоставлен организаторами
- Методы обучения: оптимизаторы, функции потерь
- Оценка: 78%
